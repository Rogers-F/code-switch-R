# Code Switch v2.7.2

## 更新亮点
- 新增 Dashboard 页面，统一顶部导航与页面布局，提升可用性与一致性。
- 配置备份/导入（ZIP）：支持可选包含 secrets 与数据库（`app.db`），便于迁移与回滚。
- MCP 卡片与 Tab 布局优化，日志/主页面结构整理，整体 UI 更清爽。
- 自动更新/发布产物命名兼容：macOS 支持 `simonswitch-macos-*.zip`（并保留对旧命名的兼容）。

## 详细变更
### UI / 交互
- 统一页面样式、间距与组件布局；优化多页面一致性与信息密度。
- 新增 `Dashboard` 页面：聚合核心状态与入口，简化日常操作路径。
- 调整窗口默认与最小尺寸为 `1280x800`（避免窄窗口导致布局挤压）。

### 配置与数据
- 新增配置备份服务：导出/导入配置包，导入后可刷新 Gemini Providers 与 Prompts 的内存状态。
- 导入时默认不包含数据库；只有显式开启 `ImportDatabase` 才会写入 `app.db`（可能覆盖本机历史）。

### 升级注意
- 若你从旧版本升级：旧的配置目录可能为 `~/.codex-swtich/`（历史拼写错误）。建议升级前先手动备份该目录（尤其是 `app.db`）。
